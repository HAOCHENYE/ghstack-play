# ghstack-play

A playground repository for experimenting with [ghstack](https://github.com/ezyang/ghstack) and GitHub workflows.

## Contents

- `.github/workflows/` - GitHub Actions workflow configurations
- `a.sh`, `b.sh`, `c.sh` - Shell scripts

## Getting Started

Clone the repository:

```bash
git clone https://github.com/HAOCHENYE/ghstack-play.git
cd ghstack-play
```
